## Issues Faced and Solutions

### Issue 1: Git not installed (Git: Clone not visible)
- Problem: The "Git: Clone" option was not showing in Cursor, so I couldn’t clone the repository.
- Solution: Found out that Git was not installed. Installed it from the official website and restarted Cursor.

### Issue 2: Confusion while opening repository in Cursor
- Problem: Initially unsure how to connect my GitHub repository with Cursor.
- Solution: Explored different approaches and understood how cloning works, and also learned the manual method as a backup.

### Issue 3: Git commit error (user.name & user.email not set)
- Problem: While committing changes, Git showed an error asking to configure user identity.
- Solution: Used terminal commands to set user.name and user.email globally.

### Issue 4: Understanding commit vs push
- Problem: Was confused between committing changes and actually uploading them to GitHub.
- Solution: Learned that commit saves changes locally, while push/publish sends them to GitHub.

### Issue 5: Extension selection confusion
- Problem: Saw multiple Claude-related extensions and wasn’t sure which one to install.
- Solution: Followed the exact naming given in the instructions and installed "Claude Code".
