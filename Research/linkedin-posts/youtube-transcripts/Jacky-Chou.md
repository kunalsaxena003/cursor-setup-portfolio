#Transcript
In this video, the speaker explains how to rank number one locally using a budget of $100, emphasizing that this amount is usually enough to purchase two essential SEO gigs. The focus of the discussion is on local SEO, with the claim that most people overcomplicate the process when, in reality, a simple and low-cost approach can be effective.

The speaker shares some personal performance metrics, including earnings from different sources such as Amazon and ad revenue platforms, while briefly acknowledging fluctuations in site performance. He then transitions into the core strategy, which revolves around purchasing cheap citation services and using private blog networks (PBNs) to boost rankings.

The first step involves buying citation backlinks. For example, the speaker mentions purchasing 350 citations for around $60, highlighting how inexpensive these services can be. These citations are then submitted for indexing using an indexing tool. The next step is to “tier” these citations using PBN links, which means building additional backlinks that point to the citation pages instead of directly linking to the main website. This indirect linking strategy is presented as a way to reduce risk while still improving rankings.

The process involves copying all citation URLs and using them in a spin-tax format, which allows for variation in anchor text and keywords. The main keyword example used is “corporate catering London.” These keywords are inserted into the PBN content so that the generated articles are contextually aligned with the target search terms.

Additionally, the speaker recommends embedding structured data into the backlinks. This includes adding business information such as name, address, phone number, and website URL, along with a Google Maps embed. The goal is to create content that appears more legitimate and relevant to local SEO signals. The embed code is generated using tools like ChatGPT and then inserted into blog posts or pages, often tested in a WordPress environment and validated using schema testing tools.

A key tactic mentioned is making URLs non-clickable within the embedded content to avoid potential penalties while still allowing search engines to recognize the information. The speaker demonstrates validating the structured data to ensure it correctly displays business details, reinforcing local SEO signals.

Finally, the process concludes with ordering PBN links and instructing the provider to include the generated embed code in each article. The speaker suggests directly communicating with service providers to ensure proper implementation. Overall, the method combines low-cost citations, PBN tiering, keyword optimization, and structured data embedding to improve local search rankings.
